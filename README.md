# Reaction Timer - 3D Edition

A stylish, interactive reaction time game built with HTML5 and Three.js that tests your reflexes with a modern 3D interface.

## Features

- **3D Background Animation**: Dynamic 3D visuals powered by Three.js
- **Responsive Design**: Beautiful UI that works on all screen sizes
- **Reaction Testing**: Measure your reaction time accuracy
- **Real-time Feedback**: Immediate response to your performance
- **Modern Aesthetics**: Glassmorphism design with smooth animations

## How to Play

1. Open `index.html` in your web browser
2. Wait for the signal to indicate you should click
3. Click as quickly as possible when the signal appears
4. Your reaction time will be displayed
5. Try to beat your personal best!

## Technology Stack

- **HTML5**: Markup structure
- **CSS3**: Modern styling with backdrop filters and gradients
- **JavaScript**: Game logic and interactivity
- **Three.js**: 3D background animations
- **Google Fonts**: Inter and JetBrains Mono typefaces

## Getting Started

Simply clone or download this repository and open `index.html` in any modern web browser. No build process or dependencies required!

## Browser Compatibility

Works best on modern browsers that support:
- CSS backdrop-filter
- WebGL (for Three.js)
- ES6 JavaScript
